# Changelog

## 0.1.0 - v1 readiness scaffold

Initial v1-ready scaffold composed across Layers 0-10:

- Foundation monorepo and project adapter model.
- PostgreSQL-backed durable memory store.
- Memory candidate ingestion lifecycle.
- Planned, ranked, budgeted, and audited context packs.
- Compiled Markdown/JSON handoffs.
- Governance and evaluation hooks.
- Model-agnostic LLM gateway with mock/local/provider adapter paths.
- Artifact memory and repository indexing.
- CLI workflows for software-development sessions.
- Admin UI / memory browser.
- Production readiness scripts, runbooks, health/readiness endpoints, and release checklist.
