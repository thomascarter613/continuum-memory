# Project Charter

## Product

Continuum Memory is a persistent memory, context, and handoff control plane for LLM-assisted software development.

## Mission

Make software-development work survive across chats, models, tools, agents, and sessions by separating durable memory from disposable conversations.

## Core promise

A developer should be able to stop work in one chat and resume in another chat with enough project state, decisions, constraints, and next actions to continue productively.

## Non-goals for Layer 0

- Full autonomous coding agent
- Production-grade persistence
- Hosted SaaS
- Complete vector retrieval
- Complete policy enforcement
- Complete UI

## Layer 0 goals

- Establish repository architecture.
- Define memory taxonomy.
- Define APIs and schemas.
- Provide local API shell.
- Provide handoff artifact format.
- Provide project adapter template.
