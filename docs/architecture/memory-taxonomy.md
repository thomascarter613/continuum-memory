# Memory Taxonomy

## Working Memory

Temporary state for the current run, task, session, or turn.

## Episodic Memory

What happened. Examples: messages, tool calls, command results, debugging attempts, accepted recommendations.

## Semantic Memory

Stable facts, preferences, project state, definitions, and domain knowledge.

## Procedural Memory

How to do things. Examples: runbooks, workflows, checklists, prompts, scripts, skills, task procedures.

## Decision Memory

ADR-like records that preserve context, alternatives, rationale, consequences, and reversibility.

## Artifact Memory

Files, generated outputs, diagrams, logs, reports, datasets, and attachments.

## Handoff Memory

Portable context packs that allow work to resume across chat/session/model boundaries.
