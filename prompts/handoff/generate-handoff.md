# Handoff Generation Prompt

Create a portable handoff pack that allows a new chat, new model, new agent, or human developer to continue the work without needing the original transcript.

Include:

1. Objective
2. Current state
3. Accepted decisions
4. Active constraints
5. Files/artifacts changed or created
6. Commands run
7. What passed
8. What failed
9. Open questions
10. Risks
11. Next exact actions
12. Context that must be preserved

Do not include private unrelated memory.
Do not invent completed work.
Do not hide uncertainty.
