# Agent Instructions

This repository uses a Continuum Memory project adapter.

## Before starting work

1. Read `.memory/project.yaml`.
2. Read `docs/work/current-state.md`.
3. Read relevant ADRs in `docs/adr/`.
4. Check the latest handoff in `docs/handoffs/`.
5. Prefer small, verifiable changes.

## After meaningful work

1. Update `docs/work/current-state.md`.
2. Add or update ADRs if decisions changed.
3. Generate a new handoff in `docs/handoffs/`.
4. Record exact commands run and results.

## Never commit

- raw chat transcripts
- private personal memory
- secrets
- embeddings
- vector indexes
- database volumes
- unreviewed sensitive logs
